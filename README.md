# 🍌 Banana Ripeness Detection – Roboflow

Eine KI-gestützte Web-App zur Erkennung des Bananreifegrades 
mithilfe eines vortrainierten Roboflow-Modells.

## Modell
- Roboflow: banana-ripeness-classification
- Klassen: unripe, freshunripe, freshripe, ripe, overripe, rotten

## Setup
1. Streamlit Secrets setzen:
ROBOFLOW_API_KEY = "dein_key"
2. App starten: streamlit run app.py
